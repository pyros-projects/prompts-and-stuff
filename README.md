## Philosophy

**On LLMs:**
- They're weirder than their defaults suggest. Structured constraints produce genuinely different results.
- Weird combinations are features, not bugs. A children's story at nihilistic-awe settings? That's the point.
- Explicit beats implicit. "Write something sad" < "B=7, E=8" (sensitive and raw, tragic weight).

**On scripts:**
- If I needed it twice, it goes in the repo.
- Interactive mode for exploration, CLI mode for automation.
- Comments are for future-me who will have forgotten everything.

**On organization:**
- Folders exist. Use them.
- READMEs are mandatory. I will forget what things do.
- MIT license because sharing is caring.

## Usage

**For prompts and games:** Copy the relevant file into your LLM conversation and follow the instructions within. Most work with Claude, GPT-4, Gemini, etc.

**For scripts:** Check the README in each script folder. Usually involves installing some modules and running PowerShell.
